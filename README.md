# DragonJudge 🐉

A股情绪龙头助手 - 智能分析系统

## 核心功能

- 📊 情绪周期判断（冰点/启动/发酵/高潮/分化/退潮）
- 📈 涨停梯队追踪（连板高度、晋级率）
- 🎯 Dragon Judge 评分系统（5维度评分）
- 📋 每日复盘报告（情绪+梯队+明日策略）
- 🔔 实时预警（龙头断板、新主线启动）
